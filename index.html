EVENTS API

How to use this service API

url: /api/events

Available endpoints for EVENTS

GET /

- get all events

GET /{eventID}

- get an event by his id

POST /

- add an event

Data to pass in body request for this route

  • title: string
  • description: string
  • date: Date(YYYY-MM-DD)
  • maxSeats: number (int)
PUT /{eventID}

- edit an event

Data to pass in body request for this route

  • title: string
  • description: string
  • date: Date(YYYY-MM-DD)
  • maxSeats: number (int)

Available endpoints for RESERVATIONS

GET /{eventID}/reservations

- get all reservations for an event

POST /{eventID}/reservations

- add a reservations to an event

Data to pass in body request for this route

  • firstName: string
  • lastName: string
  • email: string
DELETE /{eventID}/reservations/{reservationsID}

- delete a reservations
